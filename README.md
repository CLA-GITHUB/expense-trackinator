# [__Expense Trackinator__](https://expense-trackometer.netlify.app/)


This app was bootstrapped by [create-react-app](https://github.com/facebook/create-react-app) and developed by [__C.O.A.L Solution__](https://coalsolutions.com/)


This is a simple application made for public use. It enables users input their transactions and track expenses made. No ads, no fee, just a simple applicatoins.

Anyone is free to contribute to this application.

The backend can be found [here](https://github.com/CLA-GITHUB/expense-trackinator-backend)
